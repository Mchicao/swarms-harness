# Provider icon sources

These SVG assets come from [theSVG](https://thesvg.org/) and are embedded in
the native Rust UI. The upstream collection is MIT licensed.

- `openai.svg`: `openai/dark.svg`
- `gemini.svg`: `google-gemini/default.svg`
- `anthropic.svg`: `anthropic/dark.svg`

- `zcode.png`: copied from the locally installed ZCode application at
  `resources/icon.png`; displayed for the `zai:coding` plan.
