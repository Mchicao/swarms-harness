# Code Of Conduct

This project follows the Contributor Covenant Code of Conduct, version 2.1.

## Our Pledge

We pledge to make participation in this project a respectful experience for everyone.

## Expected Behavior

- Be direct and constructive.
- Keep technical disagreement focused on the work.
- Respect maintainers' safety and scope decisions.
- Do not post secrets, credentials, or private user data.

## Unacceptable Behavior

- Harassment, insults, or personal attacks.
- Publishing private credentials or private logs.
- Repeatedly pushing changes that enable paid providers by default.
- Misrepresenting experimental behavior as production-ready.

## Enforcement

Maintainers may remove comments, reject contributions, or block users who violate this code.
